# Addon List (Fixed Edition)

[**Addon List**](#) brings you the convenience of in-game addon management.  
Originally created by Blizzard and backported from Classic by [**Tsoukie**](https://gitlab.com/Tsoukie/), it has a clean and intuitive layout, displaying essential information about each addon (status, version, author, memory usage). Users can easily enable or disable addons directly in-game.

This fork adds **modern UI compatibility fixes** to make it work on clients with **NineSlice-based frames** and **mixed dropdown APIs**, preventing errors from `!!!ClassicAPI`.

---

## ⋮ KEY FEATURES

- Enable or disable selected addons in-game
- Show author, version, and memory usage on hover
- Lightweight and Blizzard-like design
- Compatible with modern `NineSlice` UI frames and `!!!ClassicAPI`
- Always works. Never bugs out.

<summary>Installation Instructions</summary>

1. Extract the downloaded compressed files _(Right-Click → Extract-All)_.
2. Inside the extracted folders, locate `AddonList` and `!!!ClassicAPI`.
3. Move these folders into your client’s `Interface\AddOns\` directory.
4. Restart your WoW client.
